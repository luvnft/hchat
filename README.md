# Chatly

Chatly is an open-source social platform similar to Discord or Guilded, built using Tauri, React/TypeScript, and Shadcn/UI.

## Tech Stack

- 🐰 BunJS
- 🚀 Tauri
- ⚛️ React
- 🎨 Tailwind CSS
- 🧰 Shadcn/UI
- 🎙️ LiveKit.io
- 🕙 Timescale
